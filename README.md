#Banco 100% digital:

El Banco del País, uno de los bancos más importante de tu país. Es una institución de muchos años y actualmente está pasando por un proceso de transformación digital. Como parte de este proceso, han implementado un área de innovación que quiere llegar de una mejor manera a los usuarios más jóvenes. Por lo que se han propuesto crear un banco 100% digital dirigido para millenials.
